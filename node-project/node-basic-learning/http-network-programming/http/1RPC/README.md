# RPC

RPC（[Remote Procedure Call，远程过程调用](https://blog.csdn.net/u011748319/article/details/107663088）是一种通过网络从远程计算机请求服务的协议，通俗来讲，一次RPC调用就是，客户端A把需要请求的服务名，以及服务对应的参数通过网络传到服务端，由服务端执行服务后，将结果通过网络返回给客户端的过程
